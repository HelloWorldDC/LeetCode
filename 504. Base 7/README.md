# 504. Base 7

Given an integer `num`, return *a string of its **base 7** representation*.

 


**Example 1:**

>**Input:** num = 100  
**Output:** "202"  

**Example 2:**

>**Input:** num = -7  
**Output:** "-10"  
 

**Constraints:**

<li><code>-10<sup>7</sup> &lt;= num &lt;= 10<sup>7</sup></code></li>
